欢迎进入HTML世界

1. HTML基础概念及引入

  1. html(Hyper Text Markup Language):超文本标记语言
  2. 网页：一个html文件
  3. 网站：许多网页的集合
  4. 常见浏览器：IE，Edge,火狐（firebox), 谷歌（Chrome), Safari 和 Opera等
  5. web标准：由W3C(万维网联盟)和其他标准化组织制定的一系列标准的集合
  6. Web标准的构成:
    1. 结构(Structure)—网页元素整理和分类—html
    2. 表现(Presentation)—网页元素的版式颜色大小等设计—CSS
    3. 行为（Behavior)—网页模型的定义及交互的编写—javascript

2. HTML基础语法

  1. html由标签组成的大致框架，标签又可分为单标签和双标签
  2. 标签关系：并列和包含

3.HTML文件多余部分作用

  1. <!DOCTYPE html>:告诉浏览器用哪种html类型来显示网页

  2. <html lang="en" >:用定义当前文档所显示的语言为英语(zh-CN为中文)

  3. ：标签的 charset属性规定html文档采用的字符编码 "UTF-8" :万国码

4.HTML常用标签

  1. 标题标签：<h1> - <h6>

  2. 段落标签：<p> </p>

  3. 换行标签：<br />

  4. 文本格式化标签：见上图代码（加粗，倾斜，删除线，下划线）

  5. 大小盒子标签：<div></div>：用来布局，一行放一个，大盒子

    ​ <span></span>：用来布局，一行放多个，小盒子

  6. 图像标签：

    1. <img src="images/Carry.jpg" border = "15 title="我最喜欢的球员" alt="carry" height = "500" />

      • src:图片路径，必须属性。
      • alt:替换文本。图像不能显示时显示文字
      • title:提示文本。鼠标放到图像上显示文字
      • width:像素，设置图像宽度（height高度，broder边框粗细)

  7. 注释标签

    • 快捷键：ctrl+\

    • 常用符号化标签：空格键：&nbsp

      ​ 小于号：&lt

      ​ 大于号：&gt

    8.与路径有关知识
    • 相对路径：以文件所在位置为基础列出来图像的位置(/)
    1. 同一级路径：<img src = "sougou.gif" />
    2. 下一级路径: <img src = "images/sougou.gif" />
    3. 上一级路径：<img src = "../sougou.gif" />
    • 绝对路径：是指目录下的绝对位置，直接到达目标位置，通常从盘符开始,或者网络上的绝对地址()

    ​ <img src = "C:\Users\24362\Pictures\Camera Roll\abs.png" />

    9.超链接标签

    • 语法格式：<a href=" 跳转目标" target="跳转方式，_self,_blank">文本或图像</a>
    • href为链接的url地址，target为窗口的弹出方式

    10.链接的分类

    • 外部链接：<a href="http://wwww.baidu.com">百度</a>

    • 内部链接：<a href="index.html">目录</a>

    • 空连接：<a href="">空链接</a>

    • 下载链接：href=”文件或者安装包的地址”

    • 网页元素链接：在网页中的各种元素的链接

    • 锚点链接（！！！）：<a href="#名字">点击此处返回目录</a>

      ​ <h1 id="名字">目录</h1>
